# UltraBot Video Detection System

## Overview
This project implements a video object detection and trajectory analysis pipeline using YOLOv8 and LSTM models.

The system performs:
- Object detection on video frames
- Temporal sequence modeling using LSTM
- Trajectory visualization
- Prediction output generation
